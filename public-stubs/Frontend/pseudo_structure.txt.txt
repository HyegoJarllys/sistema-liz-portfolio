frontend/

    app/
      ui.py (or App.tsx)
        - main UI layout
        - input box, submit action
        - response rendering

    api/
      client.py (or api.ts)
        - backend base URL (placeholder)
        - POST /chat wrapper
        - timeout + retry policy (lightweight)

      session.py

        - local persistence (cookie/localstorage/session)

      settings.example
        - placeholder env keys (no secrets)
        - baseUrl defaults to <backend-host>
